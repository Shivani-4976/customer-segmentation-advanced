# 🧠 Customer Segmentation (Advanced)

## 📌 Problem Statement

Segment customers based on income, spending behavior, and age to help businesses improve marketing strategies.

## 📊 Dataset

* Mall Customers Dataset (Kaggle)

## ⚙️ Approach

* Feature Engineering (Spending to Income ratio)
* Data Scaling (StandardScaler)
* PCA for dimensionality reduction
* KMeans Clustering
* Silhouette Score for evaluation

##  Results

* Identified 5 customer segments
* Successfully grouped customers based on behavior

##  Cluster Insights

* Cluster 1 → Premium customers (high income, high spending)
* Cluster 3 → Target customers (high income, low spending)
* Cluster 2 → Impulsive buyers (low income, high spending)
* Cluster 0 → Low-value customers
* Cluster 4 → Young trend-driven customers
